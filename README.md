# FACT in AI - Group 5

This repository contains the code and documentation for Group 5's project in the
FACT in AI course. The project focuses on [Differentially Private Fine-Tuning of
Diffusion
Models](https://openaccess.thecvf.com/content/ICCV2025/papers/Tsai_Differentially_Private_Fine-Tuning_of_Diffusion_Models_ICCV_2025_paper.pdf).

## Installation

The directory `dp_lora` contains the original code from the paper's
authors. To set up the Python environment using Anaconda, run the script: 
```
dp_lora/setup_env.sh
```
